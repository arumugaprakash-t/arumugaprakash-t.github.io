# arumugaprakash-t.github.io

Personal website powered by Jekyll with the al-folio theme.

## Setup

This site uses Jekyll and GitHub Actions for automatic deployment to GitHub Pages.

### Local Development (Optional)

If you want to preview the site locally, you'll need Ruby installed:

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://localhost:4000` to see your site.

### Deployment

The site automatically deploys to GitHub Pages via GitHub Actions whenever you push to the `main` branch.

## Configuration

- **_config.yml**: Main site configuration
- **_pages/about.md**: Homepage content
- **Gemfile**: Ruby dependencies
